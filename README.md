# BinChilling_preprocessor
Use pug/Scss

## 初始環境建立
``` 
npm install               // 安裝專案dependencies
npm i -g parcel-bundler   // 全局安裝 parcel

parcel aboutUs.pug  // 會出現  http://localhost:1234 ，Ctrl + click 即可開啟頁面

```
